# Client Onboarding

Use this checklist for each pilot contractor.

## Required Client Details

- Business name
- URL slug
- Brand color
- Optional logo URL
- Notification email
- Phone number
- Website URL
- Service area
- Project types

## Setup

1. Add or update the client in Supabase `clients`.
2. Confirm `is_active = true`.
3. Open `/intake/[clientSlug]`.
4. Submit a test inquiry.
5. Confirm contractor email delivery.
6. Confirm homeowner confirmation email delivery.
7. Share the intake URL with the contractor.

## Launch Link Placements

- Existing website contact page
- Google Business Profile
- Email follow-up templates
- Social profiles
- Paid ad landing page buttons

## First Pilot Review

After the first 20-50 submissions, review:

- Where homeowners get stuck
- Which fields are often missing
- Whether AI summaries are useful
- Whether fit scoring matches contractor expectations
- Whether email deliverability is healthy
